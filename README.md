# readmecraft

A lightweight CLI tool to generate clean, structured README files.

## Features
-
 Fast
-
 Lightweight
-
 No config
-
 Clean template

## Installation
```bash
npm i -g readmecraft
#Usage
readmecraft
# or
readmecraft --out README.md
# non-interactive
readmecraft --no-prompt --force
